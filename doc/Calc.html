Calc - Scientific Java Calculator


Do you miss a real calculator for your java-enabled phone (especially the T610)? Do you sometimes need to calculate the sine of an angle but left your real calculator at home? Calc is a Java application for your phone that gives you more than you probably need.

Function

Calc works much like a good old HP calculator with RPN logic, but the stack has 16 elements and you can see many of the elements on the stack simultaneously. I have chosen to use RPN logic (Reverse Polish Notation), because operating it requires less keys, you need only ENTER instead of "(" and ")". Calc has 63 bits internal precision, considerably more than normal calculators and computers, approximately 19 decimal digits accuracy. The exponent is 31 bits, which allows for numbers up to 4.197E+323228496. Handy when you want to calculate the factorial of 40000000.

Keys

To be able to navigate efficiently through function menus, Calc can only be operated on a phone with a 5-way "joystick" or equivalent arrow keys, plus a "clear" key, such as the Sony-Ericsson T610. The keys are mapped as follows:
  • 0-9: The number keys are used to enter numbers and as shortcuts when executing commands that take a number argument. In hexadecimal mode, you can enter the digits a-f by holding the keys 1-6 down.
  • *: This key is used to enter the decimal point.
  • #: This key is used to enter a minus sign or an "E" for exponent.
  • clear, 'c': This key deletes an entered number, clears the lowest element on the stack, and backtracks through menus.
  • ENTER: This key enters the currently edited number onto the stack. If no number is being edited, the lowest element on the stack is duplicated.
  • + : This key adds the two lowest elements on the stack.
  • Joystick: Pressing the middle of the joystick activates the menu. Pressing up/down/right/left on the joystick activates the corresponding sub-menu directly. Sub-menus are entered by further navigation with the joystick until an actual operation is activated.

Operations

In the following operations, x represents the lowest element on the stack, y represents the element above x. In operations with two inputs, such as +,-,* and /, the x and y stack elements are removed from the stack and the result is placed on the stack instead, as the new x. In operations with one input, such as ex, x is removed from the stack and the result is placed on the stack instead. To add 3 and 7, for example, one would press the following sequence of keys: 3, ENTER, 7, +. The result, 10 will now be the lowest element on the stack, ready for further calculations.

The following operations are available through the menu system:

Basic

  • - : Subtracts x from y
  • * : Multiplies x and y
  • / : Divides y by x
  • +/- : Changes sign of x

Math

  • Simple
    • abs: Makes x positive
    • x2: Calculates the square of x
    • 1/x: Calculates the reciprocal of x
    • sqrt(x): Calculates the square root of x
  • Pow
    • ex: Calculates the natural exponential
    • ln: Calculates the natural logarithm
    • yx: Calculates y to the power of x
    • x-root(y): Calculates the x'th root of y. Works with negative values of y if x is an odd integer.
  • Pow2,10
    • 2x: Calculates 2 to the power of x
    • log2: Calculates base-2 logarithm of x
    • 10x: Calculates 10 to the power of x
    • log10: Calculates base-10 logarithm of x
  • Pol
    • atan2: Calculates the angle of y/x, in the range -pi to pi (or -180 to 180 in DEG mode)
    • hypot: Calculates sqrt(x2+y2)
    • r->p: Convert rectangular coordinates (x,y) to polar (x=radius, y=angle)
    • p->r: Convert polar coordinates (x=radius, y=angle) to rectangular (x,y)
  • Prob
    • P y,x: Calculates permulations of x elements taken from y
    • C y,x: Calculates combinations of x elements taken from y
    • x! : Calculates the factorial of x. For non-integral values, calculates Gamma(x+1)
    • random: Calculates a random number between 0 and 1

Trig

  • Normal
    • sin: Calculates the sine of x
    • cos: Calculates the cosine of x
    • tan: Calculates the tangent of x
  • Arc
    • asin: Calculates the arc sine of x
    • acos: Calculates the arc cosine of x
    • atan: Calculates the arc tangent of x
  • Hyp
    • sinh: Calculates the hyperbolic sine of x
    • cosh: Calculates the hyperbolic cosine of x
    • tanh: Calculates the hyperbolic tangent of x
  • Archyp
    • asinh: Calculates the hyperbolic arc sine of x
    • acosh: Calculates the hyperbolic arc cosine of x
    • atanh: Calculates the hyperbolic arc tangent of x
  • More
    • RAD: Sets radians as measure of angles (2*pi is full circle)
    • DEG: Sets degrees as measure of angles (360 is full circle)
    • ->RAD: Converts from degrees to radians
    • ->DEG: Converts from radians to degrees
    • pi: Enters the number pi

Special

  • Stack
    • LAST x: Recall value of x before last operation
    • x<->y: Exchange x and y elements of the stack
    • x<->st#: Exchange x with any element on the stack
    • RCL st#: Copy any stack location
    • clear: Clear the stack
  • Mem
    • STO: Store x in memory location
    • STO+: Add x to memoty location
    • RCL: Recall memory location
    • x<->mem: Exchange x with any memory location
    • clear: Clear memory
  • Stat
    • SUM+ : Add x and y to statistics
    • SUM- : Subtract x and y from statistics
    • Result
      • x,y: Calculate mean x and mean y values
      • sx,sy: Calculate standard deviation of x and y values
      • L.R.: Linear regression, y=intersection, x=slope
      • y*,r: Calculate estimated y for current x, and correlation coefficient
      • n: Recall number of statistics entered
    • Regs
      • SUMx: Recall sum of x'es
      • SUMx2: Recall sum of squared x'es
      • SUMy: Recall sum of y's
      • SUMy2: Recall sum of squared y's
      • SUMxy: Recall sum of xy product
    • clear: Clear statistics
  • Int
    • round: Round x to nearest integer
    • trunc: Remove fractional part of x
    • ceil: Round x towards positive infinity
    • floor: Round x towards negative infinity
    • frac: Remove integer part of x
  • Time
    • ->DH.MS: Convert x from hours to date, hours, minutes and seconds. Date format is yyyymmddHH.MMSS. If x<24000, it is converted to the format ddddHH.MMSS, where dddd is the number of days.
    • ->H: Convert date, hours, minues and seconds to hours. If yyyy is zero, format is understood as ddddHH.MMSS, where dddd is number of days.
    • date: Get current date in DH.MS format
    • time: Get current time in DH.MS format
    • DH.MS+: Add x and y as date, hours, minues and seconds

Mode

  • Number
    • normal: Set normal number mode, shows all digits but removes trailing zeros
    • FIX: Set fix number mode, shows a fixed number of digits after the decimal point
    • SCI: Set sci number mode, shows a fixed number of digits in scientific format
    • ENG: Set eng number mode, shows a fixed number of digits after the decimal point in engineering mode.
  • Sepr
    • point
      • . : Display the decimal point as a period
      • , : Display the decimal ponit as a comma
      • keep: Keep the decimal point even if no fraction is shown
      • remove: Remove the decimal point if no fraction is shown
    • thousand
      • . or , : Display period or comma as thousands separator
      • space: Display space as thousands separator
      • ' : Display ' as thousands separator
      • none: Display no thousands separator
  • Base
    • DEC: Set decimal (base-10) number mode
    • HEX: Set hexadecimal (base-16) number mode
    • BIN: Set binary (base-2) number mode
    • OCT: Set octal (base-8) number mode
  • Font
    • small: Set small font
    • medium: Set medium font
    • large: Set large font
