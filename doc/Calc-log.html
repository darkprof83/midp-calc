Calc - Scientific, Statistical & Financial Java Calculator

Changelog:

Version 1.11, 2004-02-23

  • Made "int" submenu accessible in hex/oct/bin mode
  • Hex number "1.fffffffffffffffc" displays correctly as "2"
  • Faster and more accurate division routine
  • Statistics log (used in "draw") now stores 64 x/y pairs and removes correctly on Σ-
  • Using CRC rather than linear congruential generators in random number generator. RNG now passes ent test
  • Released Real.java class under GPL

Version 1.08, 2004-01-17

  • Added percent command to basic menu
  • Added undo command to stack menu (removed x<>st#)
  • atan2(0,0) correctly returns 0

Version 1.02, 2003-12-19

  • yx faster and more accurate if x is integer
  • Added financial functions y%*x and y%/x
  • Added financial mode for beginning and end of month payments
  • Financial solving works properly with ir% = 0
  • Setup dialog to query about keyboard layout

Version 1.00, 2003-11-29

  • First public version
